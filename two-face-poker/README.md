# 양면 포커 (Two-Face Poker)

1대1 온라인 실시간 웹 게임. Node.js + WebSocket.

## 실행

```bash
npm install
npm start
```

브라우저 2개에서 `http://localhost:3000` 접속 → 한 명이 "방 만들기" → 코드 공유 → 상대가 입장하면 자동 시작.

혼자 연습하려면 로비에서 **"🤖 AI와 대전"** 버튼으로 즉시 시작할 수 있습니다. AI는 카드 가치 기반으로 면 선택·레이즈·콜·폴드를 판단하고, 양면이 모두 높으면 양면 배팅도 시도합니다.

다른 기기와 플레이하려면 같은 네트워크에서 `http://<내IP>:3000`으로 접속하거나, 서버를 호스팅(Render, Railway 등)하면 됩니다. `PORT` 환경변수 지원.

## 규칙

- 카드는 양면: **White=짝수(2~10)**, **Black=홀수(1~9)**. 자기 카드의 양면만 볼 수 있음.
- 시작 코인 **40개**. 첫 선은 랜덤, 이후엔 **진 사람이 선**.
- 선부터 면을 선택하며 **기본 배팅 1코인** 자동 지불. White / Black 중 한 면에만 배팅.
- 이후 선부터 **체크 / 레이즈 / 콜 / 폴드** 진행. 콜이 나오거나 둘 다 체크하면 공개.
- 배팅한 면의 숫자가 높은 쪽이 팟 획득. **무승부면 팟은 다음 게임으로 이월**.
- **양면 배팅**: 두 면 모두 상대를 이길 자신이 있으면 양쪽에 같은 코인을 걸 수 있음(모든 지불 2배). 두 면 모두 상대 숫자보다 높으면 팟 + **보너스 10코인**을 상대에게서 획득. 실패하면 상대가 팟 획득.
- 상대 코인을 모두 가져오면 최종 승리.

## 파일

- `server.js` — 게임 서버 (방 관리 + 전체 게임 로직)
- `public/index.html` — 클라이언트 (단일 파일)
- `test.js` — 자동 시뮬레이션 테스트 (`node server.js` 실행 중에 `node test.js`)
- `test-ai.js` — AI 대전 자동 테스트
